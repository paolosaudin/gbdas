The program doesn't start-up Top « Home


Verify the parameters in the 'gbdas.conf' file.
  • host=myHost
             Host name (usually localhost)
  • user=xxx
             MySql user name
  • passwd=xxx
             MySql password
  • dbname=xxx
             MySql db name
  • baudrate=9600
             Adam 4017 module speed connection
  • autostart=0
             Run automatically at startup
